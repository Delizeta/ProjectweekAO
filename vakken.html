Nederlands
Nederlands wordt gegeven bij elke opleiding als basisvak. Op de opleiding AO krijg je anderhalf uur Nederlands per week. Aan het eind van de periode wordt er een toets afgenomen. Een periode duurt 10 weken. Je Krijgt in totaal 4 toetsen. De toetsen zijn lees en luisteren, spreekvaardigheid en schrijfvaardigheid.

Engels
Engels wordt gegeven bij elke opleiding als basisvak. Op de opleiding AO krijg je anderhalf uur E per Engels week. Aan het eind van de periode wordt er een toets afgenomen. Je Krijgt in totaal 4 toetsen per jaar. De toetsen zijn lees en luisteren, spreekvaardigheid en schrijfvaardigheid.

Rekenen
Rekenen wordt gegeven bij elke opleiding als basisvak. Op de opleiding AO krijg je anderhalf uur Rekenen per week. Aan het eind van de periode wordt er een toets afgenomen. Je Krijgt in totaal 4 toetsen. Je hebt verschillende hoofdstukken met allemaal andere stof.

Burgerschap Burgerschap wordt gegeven bij elke opleiding als basisvak. Op de opleiding AO krijg je anderhalf uur Burgerschap per week. Aan het eind van de periode wordt er een toets afgenomen. Je Krijgt in totaal 4 toetsen. De hoofdstukken zijn politiek, cultuur, economy en seksuele voorlichting.

HTML
HTML is een afkorting van Hyper Text Markup Language. Zoals de term al zegt is het een "opmaak taal", d.w.z. Het vertelt je browser d.m.v. HTML tags hoe de pagina weergegeven moet worden op het scherm. HTML is geen programmeertaal zoals vaak gedacht wordt HTML tags zijn als lego bouwstenen. Met die bouwstenen kun je een gebouw bouwen. Met HTML een website.

Java
Wat is Java? Java is een objectgeoriënteerde (OO) taal. Dit wil zeggen dat je alles beschrijft zoals in het dagelijkse leven. Je hebt objecten, en deze objecten bezitten eigenschappen (variabelen) en kunnen handelingen verrichten (methoden). Meestal schermt men in Java de variabelen en de implementatie van elk object af van de buitenwereld.

Javascript
JavaScript is de programmeertaal die het meest gebruikt wordt om zogeheten dynamische webpagina's te maken. Het is dus een taal die voor interactie met de gebruiker zorgt. Bij HTML en CSS kun je er niet voor zorgen dat er iets gebeurt met de webpagina als de gebruiker op een toets drukt, ergens op klikt of iets anders doet. Daar is JavaScript voor.

php
PHP is in tegenstelling tot HTML dynamisch. Je kunt het zien als aanvulling op of onderdeel van HTML. HTML zorgt voor de opmaak, PHP zorgt voor de verwerking van gegevens. Stukjes PHP-code kunnen worden opgenomen in de HTML-pagina’s. PHP is dus ook geen programmeertaal maar een scripttaal

IT Essential
IT Essentials is een vak waar je alle hardware onderdelen behandelt wordt. Ook worden alle software componenten van de pc behandelt worden.

Datacom
Deze 'data' zijn digitale gegevens. Dit kunnen zowel complete documenten en afbeeldingen zijn als alfanumerieke velden. Datacommunicatie draait om het uitwisselen van deze digitale gegevens via een netwerk. De computers in een kantoor of een ander utiliteitspand kunnen op verschillende plaatsen zijn neergezet.

Eduscrum
eduScrum is een actieve samenwerkingsvorm, waarmee leerlingen in teams opdrachten maken volgens een vast ritme. Zij plannen en bepalen zelf hun activiteiten en houden de voortgang bij. De leraar 'bepaalt' de opdrachten, coacht en geeft raad.

Open source Open source software is software waarvan de broncode is gepubliceerd en vrij beschikbaar is voor het publiek. Iedereen kan op die manier vrij kopiëren, aanpassen en verspreiden zonder kosten aan auteursrechten en toeslagen.
